Modul Performanta > Actiuni divizii

Fereastra [Actiuni divizii] este compusa dintr-o zona de filtrare si o lista de actiuni.

 

Zona de filtrare contine urmatoare campuri, care permit cautarea:

•Anul - un anumit an
•Cod divizie - codul de divizie
•Divizie - numele de divizie
•Cod departament  - cod departament
•Departament - nume departament
•Dimensiune - o anumita dimensiune
•Status - un anumit status

 

Pentru a cauta o actiune se aleg filtrele dorite si se apasa pe butonul [Cauta]

 

 

Lista de actiuni contine urmatoarele coloane:

•Divizie - numele diviziei
•Departament - numele departamentului
•Dimensiune - numele dimensiunii
•Actiune - numele actiunii
•Status - statusul actual al actiunii
•Manager - numele managerului care a adaugat actiunea

 

Implicit lista contine doar actiunile care au fost adaugate de  catre managerii de departament.

Fiecare manager de departamant poate modifica actiunile pe care le-a definit prin apasarea pe numele actiunii.

 

Lista permite si ordonarea ascendenta si descendenta pentru fiecare coloana.

Pentru a exporta datele din lista intr-un fisier de tip Excel se poate folosi butonul [Exporta].

 

Pentru a putea tipari continutul listei se foloseste butonul [tiparire]. Pentru a putea tipari tot continutul listei, indiferent de numarul de pagini, se foloseste butonul [tipareste tot].
