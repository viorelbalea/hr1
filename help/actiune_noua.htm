Modul Performanta > Plan actiuni

Fereastra [Actiune noua] contine urmatoarele elemente:

Campurile:

•Anul - anul pentru care se adauga actiunea.
•Dimensiunea - dimensiunea
•Actiunea - descrierea actiunii
•Deadline - data la care trebuie finalizata actiunea
•Status - starea actiunii ( implicit pentru o actiune noua este "<%ST_PLANNED%>" )
•Comentariu - un comentariu asupra actiunii

 

Butoanele:

•[Salveaza] - salveaza informatiile completate
•[Inapoi] - intoarcerea, fara a salva informatiile, la lista Plan actiuni
