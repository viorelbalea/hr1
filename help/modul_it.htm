Ferastra [Modul IT], folosita pentru a specifica drepturile in aplicatiile companiei pentru fiecare angajat, contine urmatoarele:

•In partea din stanga sunt listate toate Aplicatiile si modulele (vezi Administrare - Aplicatii)
•Pentru fiecare modul in parte exista 3 coloane:
oR/W - Read / Write  (Drepturi si de citire si de modificare in modul),
oR - Read ( persoana poate doar sa citesca, fara sa poata modifica )
oW - Write ( doar drepturi de scriere in modul)

 

In partea stanga a ferestrei se poate specifica daca persoanei respective i-au fost atribuite sau i-au fost incredintate:

PC, Laptop, Imprimanta, Altele sau daca persoana o are acces la Internet: Full sau limitat, la email etc.

 

Pentru a se putea salva informatiile completate in fereastra se foloseste butonul [Salveaza] din josul ferestrei.
