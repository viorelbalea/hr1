Modul Performanta > Plan actiuni

Fereastra de [Plan Actiuni] contine lista de actiuni definite de managerul care este autentificat in aplicatie.

 

Deasupra liste de actiuni se gaseste campul folosit pentru selectarea anului.

 

Lista de actiuni contine urmatoarele informatii:

•[Modifica] - afiseaza fereastra de modificare a actiunii
•Dimensiune HCM - numele dimensiunii
•Actiune - descrierea actiunii
•Deadline - data la care trebuie finalizata actiunea
•Status - statusul curent al actiunii
•[Istoric] - afiseaza fereastra de istoric a actiunii
•Comentariu - ultimul comentariu adaugat pentru actiune

 

In partea de jos a listei se gasesc urmatoare butoane:

•[Actiune noua] - se foloseste pentru a deschide fereastra de Adaugare actiune noua
•[tiparire] - se foloseste pentru a tipari continutul listei.
•[Export] - se foloseste pentru a exporta continutul listei intr-un fisier de tip Excel
•[Inapoi] - se foloseste pentru intoarcerea la fereastra Actiuni divizii

 

Lista permite si ordonarea ascendenta si descendenta pentru fiecare coloana.
